This Student Record Management System is a robust Flask-based application designed to streamline educational data handling through the following core features:

Comprehensive Data Management: Full CRUD (Create, Read, Update, Delete) functionality for student records, persisted via SQLAlchemy and an SQLite database.
Dual Authentication System: Secure user access through JWT-based local registration/login and integrated Google OAuth for streamlined third-party authentication.
Intelligent AI Assistant: An NLP-driven chatbot using spaCy and fuzzy matching to interpret natural language queries regarding student performance, class counts, and summaries.
Advanced Analytics: A dedicated analytics engine that calculates real-time metrics such as grade distributions, class averages, and performance extremes (highest/lowest marks).
Production-Ready Security: Infrastructure support for HTTPS, CORS policy enforcement, and secure session management to protect sensitive student and user data
